# Quickstart: Agentic retrieval in Azure AI Search using Python

![Flask sample MIT license badge](https://img.shields.io/badge/license-MIT-green.svg)

This sample demonstrates the fundamentals of agentic retrieval using Azure AI Search. You create a search index, a knowledge source that targets the index, and a knowledge base that integrates an LLM for query planning and answer synthesis.

## What's in this sample

| File | Description |
|------|-------------|
| `quickstart-agentic-retrieval.ipynb` | Jupyter notebook that creates a knowledge base and runs agentic queries |
| `requirements.txt` | Python package dependencies |
| `sample.env` | Environment variable template for configuration |

## Documentation

This sample accompanies [Quickstart: Agentic retrieval using Python](https://learn.microsoft.com/azure/search/search-get-started-agentic-retrieval). Follow the documentation for prerequisites, setup instructions, and detailed explanations.
