# Quickstart: Vector search in Azure AI Search using Python

![Flask sample MIT license badge](https://img.shields.io/badge/license-MIT-green.svg)

This sample demonstrates the fundamentals of vector search, including creating a vector index, loading documents with embeddings, and running vector and hybrid queries.

## What's in this sample

| File | Description |
|------|-------------|
| `vector-search-quickstart.ipynb` | Jupyter notebook that creates a vector index, uploads documents with embeddings, and runs queries |
| `requirements.txt` | Python package dependencies |
| `sample.env` | Environment variable template for configuration |

## Documentation

This sample accompanies [Quickstart: Vector search using Python](https://learn.microsoft.com/azure/search/search-get-started-vector?pivots=python). Follow the documentation for prerequisites, setup instructions, and detailed explanations.
